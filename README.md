# Itunes Search API Android test app

○ Design Pattern: MVVM\
○ Api Client: Retrofit\
○ Dependency injection: Koin\
○ UI: Jetpack Compose\
○ Local storage: Room database\
○ Audio Player: MediaPlayer\
○ Video Player: Exo Player\
